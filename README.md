# BMI Calculator

A simple web-based BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript.  
Users can enter their height and weight to instantly calculate their BMI and see the health category.

## Features
- BMI calculation from user input
- Instant result display
- BMI category classification
- Input validation
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
